# craft-bootstrap
A basic Craft 3 Instalation. Includes Gulp for some basic task running.

- Gulp Sass
- Browser Sync
- Gulp Concat
- Gulp Rename
- Gulp Uglify
- Gulp CSS Nano
- Gulp Image Min
- Gulp Cache
- Del
- Run Sequence
- Gulp Auto-prefixer
